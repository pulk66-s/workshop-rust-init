# Exercise 01: Hello World & Basic Syntax

## Learning Objectives
- Understand basic Rust syntax
- Learn how to print to console
- Work with comments
- Run a Rust program

## Instructions

1. Open `src/main.rs`
2. Complete the TODO sections to:
   - Print "Hello, Rust!" to the console
   - Print your name using a variable
   - Print formatted output with multiple variables
   - Experiment with different println! macros

## Expected Output

```
Hello, Rust!
My name is: [Your Name]
I am learning Rust in 2026
```

## Hints

- Use `println!()` macro for printing
- Use `{}` as placeholders in format strings
- Variables are declared with `let`
- String literals are enclosed in double quotes

## Running the Exercise

```bash
cd ex01
cargo run
```
